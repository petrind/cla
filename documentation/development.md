## Prerequisite

- Node (preferrably v14.16.1)
- docker
- yarn

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# unit tests using automatic recompile
$ yarn run test:watch

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
